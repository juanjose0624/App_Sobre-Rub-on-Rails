# 💎 Ruby on Rails Database Application

A web application developed with **Ruby on Rails** as part of a university database course.

This project demonstrates the fundamentals of the Ruby on Rails framework, including the use of **Active Record**, database integration, and the MVC architecture. It also includes an overview of Ruby, Ruby on Rails, and the framework's creators as part of the academic documentation.

---

## 🚀 Features

- Database integration using Ruby on Rails
- Active Record implementation
- CRUD operations
- MVC Architecture
- Academic documentation about Ruby and Ruby on Rails
- Educational implementation based on a provided database

## ▶️ Installation

1. Clone the repository.

```bash
git clone https://github.com/juanjose0624/App_Sobre-Rub-on-Rails.git
```

2. Install dependencies.

```bash
bundle install
```

3. Configure the database.

4. Run migrations.

```bash
rails db:migrate
```

5. Start the server.

```bash
rails server
```

6. Open your browser.

```
http://localhost:3000
```

## 🎓 Academic Purpose

This project was developed for educational purposes as part of a university database course.

It demonstrates the practical use of Ruby on Rails, Active Record, and relational database management while introducing the fundamentals of the Rails ecosystem and its development philosophy.

---

## 📚 Learning Objectives

- Understand the Ruby programming language.
- Learn the Ruby on Rails framework.
- Work with Active Record ORM.
- Connect Rails applications to relational databases.
- Explore the MVC software architecture.
- Understand the history and creators of Ruby and Ruby on Rails.
